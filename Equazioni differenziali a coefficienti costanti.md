# 2° Ordine

##### Omogenee
$$ay'' + by' + cy = 0$$

Per risolvere questa equazione dobbiamo prima risolvere l'equazione di secondo grado $φ(λ) = aλ^2 + bλ + c$

##### Primo metodo
se $Δ > 0$ allora  $λ = λ_1 \ e \ λ_2$  ----->  $y=C_1e^{λ_1x} + C_2e^{λ_2x}$

##### Secondo metodo
se $Δ = 0$ allora $λ = λ_0$ -----> $y = C_1e^{λ_0x} + C_2xe^{λ_0x}$
